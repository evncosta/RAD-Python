# 🐍 RAD-Python

Bem-vindo ao repositório da disciplina de **RAD (Rapid Application Development) com Python**. 

Este espaço é dedicado para armazenar todos os códigos, atividades e projetos desenvolvidos ao longo do semestre. O objetivo principal é documentar a evolução no ecossistema Python, com foco especial na construção ágil de APIs e aplicações web.
